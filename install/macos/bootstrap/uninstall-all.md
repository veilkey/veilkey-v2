# macOS — Full Uninstall (all services)

```bash
bash install/macos/bootstrap/uninstall-all.sh
```

Runs uninstall for veil-cli, LocalVault, and VaultCenter in order.

To uninstall individually:

```bash
bash install/macos/veil-cli/uninstall.sh
bash install/macos/localvault/uninstall.sh
bash install/macos/vaultcenter/uninstall.sh
```
